vue-douban demo
===================================

预览:
1. git clone git@github.com:wittyzhang/vue-douban.git
2. npm install
3. npm run dev

涉及知识点：
1. vue-cli 搭建
2. 配合gulp编译sass
3. 引用 vue-resource 插件
4. 引入 vuex (未结合到项目中，因此src/vuex 中暂时没有用到)
